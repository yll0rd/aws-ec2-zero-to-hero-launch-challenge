**EC2 Terms:**
- **AMI (Amazon Machine Image):** A template that contains the software configuration (OS, application server, applications) required to launch an EC2 instance.
- **Instance Type:** Specifies the hardware configuration (CPU, memory, storage, networking) for an EC2 instance.
- **Key Pair:** A set of security credentials (public and private keys) used to securely connect to EC2 instances.
- **Security Group:** A virtual firewall that controls inbound and outbound traffic for EC2 instances.
